# AI EDA Dashboard

Minimal Vite + React project for the AI EDA dashboard component.

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
3. Open the URL shown in the terminal.

## Build

```bash
npm run build
```
